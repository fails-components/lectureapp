In this directories third party packages are provided that need a small patch in order to work with this package.
These packages retain their original license.
Currently only 

vision_pkg.tgz

is provided, which is a special version of @mediapipe/tasks-vision, that supports loading from inside a worker.
The repository with the source code is:
https://github.com/fails-components/mediapipe
with its esmmoduleload branch.
